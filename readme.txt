== Description ==

This plugin displays the latest posts from a Twitter account in a sidebar widget.

= Plugin temporarily not available due to closing of Twitter RSS service =



== Installation ==

1. Upload the folder 'new-twitter-widget' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Appearance->Widgets" menu and add twitter widget to sidebar.
